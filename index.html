Frequently asked questions

Find quick answers to frequent questions

  • How much does using Filetronyx cost?

    Filetronyx is free during development and only charges when you deploy your app. Then, it's only $20/mo plus usage fees. There are only 2 types of usage fees:

    1. Authorizations - when a patient or provider logs in. 10k/mo for free and then only $0.01/verification after
    2. API calls - any API call other than login. 100k/mo for free and then only $0.10/100k API calls after

  • How is patient data secured?

    Patient data is stored in an encrypted at rest electronic healthcare system meeting all HIPAA and HITECH standards. Filetronyx employees only access data when absolutely required to troubleshoot development issues following a need to know approach (e.g. an API call failed to write data correctly to a specific field).

  • How is patient data shared?

    Filetronyx is in the process of joining healthcare information exchange networks to collect and share patient information for the purpose of maintaining a single health record for a given patient, but this is 100% controlled by patient permission. By default, no sharing is allowing and it is up to the developer to give the patient the choice to opt-in and share the collected information with their healthcare providers.

  • Is anonymous patient data exposed for research?

    At this time, Filetronx does not offer de-identification APIs of the data that has been collected. That said, developers are able to ask for patients' permission to collect their data for research purposes (e.g. building a machine learning algorithm).
